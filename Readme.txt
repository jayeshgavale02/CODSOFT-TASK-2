**Installation Steps**

1. Download zip file and Unzip file on your local server.
2. Put this file inside "c:/xampp/htdocs/" .
3. Database Configuration
    Open phpmyadmin
    Create Database named travelbooking.
    Import database travelbooking.sql from downloaded folder(inside database)
4.Open payment.php and add Your Razorpay Key ID.
5. Open Your browser put inside "http://localhost/[project-folder-name]/"
